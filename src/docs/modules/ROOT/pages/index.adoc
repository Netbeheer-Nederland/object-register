= NBNL Object Register

Maintainer::
    mailto:teamsemantiek@netbeheernederland.nl[Team Semantiek] (Netbeheer Nederland)
Version::
    1.4
Release date::
    2026-07-07

The _NBNL Object Register_ is a predefined, controlled set of permissible
values used to ensure consistency and interoperability in data exchange,
validation, and interpretation. Each value in the list provides a clear
definition and may be associated with metadata such as descriptions, status, or
authoritative source.

This list is maintained by _Team Semantiek_ and is considered authoritative
within the scope of the object register.
